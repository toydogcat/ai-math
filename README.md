# AI Math 🧮

這是一個關於微積分與質數演進的小說與圖文故事集，帶你走進數學家的直覺世界。

## 📱 Progressive Web App (PWA) 安裝說明

本專案已全面支援 PWA 功能。您可以將此網頁當作應用程式安裝至您的手機、平板或電腦桌面上，隨時享受離線閱讀體驗。

### 💻 電腦版安裝方法 (Chrome / Edge 等 Chromium 瀏覽器)

1. 使用 **Google Chrome** 或 **Microsoft Edge** 瀏覽器開啟 [AI Math 網站](https://toydogcat.github.io/ai-math/)。
2. 觀察瀏覽器網址列右方，會出現一個 **「安裝應用程式」** 圖標（形狀通常為一個向下的箭頭或帶有加號的顯示螢幕）。
3. 點擊該圖標並選擇 **「安裝」**。
4. 安裝完成後，將會在桌面或開始功能表產生快捷方式，開啟後將以獨立視窗流暢運行。

### 📱 iOS / iPhone / iPad 安裝方法 (Safari)

1. 使用 **Safari** 瀏覽器開啟 [AI Math 網站](https://toydogcat.github.io/ai-math/)。
2. 點擊螢幕下方（或上方網址列旁）的 **「分享」** 按鈕（向上箭頭的方塊圖標）。
3. 向上滑動分享選單，找到並點擊 **「加入主畫面」 (Add to Home Screen)**。
4. 確認名稱為 **AI-Math** 後，點擊右上角的 **「新增」**。
5. 現在您可以直接從主畫面點擊圖標進入應用程式，享受獨立無廣告的全螢幕閱讀體驗。

### 🤖 Android 安裝方法 (Chrome)

1. 使用 **Google Chrome** 瀏覽器開啟 [AI Math 網站](https://toydogcat.github.io/ai-math/)。
2. 點擊網址列右側的 **「三個點」** 選單圖標。
3. 點擊 **「安裝應用程式」** 或 **「將網頁加入主畫面」**。
4. 點擊 **「安裝」**。
5. 稍等片刻，圖標將自動顯示在您的手機桌面上，隨時一鍵開啟讀取！
