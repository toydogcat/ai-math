彩虹與霓

我們先看看圖

彩虹就是下面

霓就是上面比較淡，倒過來 紅 橙 黃 綠 藍 靛 紫

彩虹原理：

彩虹跟霓都是陽光經過下雨後的水珠經過散射以後跑出來的紅到紫，如果在水滴裡面反射一次被我們看到就是彩虹。

霓原理：

如果在水滴裡面反射兩次然後被我們看到，就是霓，光經過反射次數越多能量就會跑走越多，所以我們看到的霓就比彩虹更淡。

從一滴水開始

上圖是一滴水陽光沿著紅色線走，根據斯乃耳定律（Snell’s law）入射角 alpha 跟 折射角 beta 會滿足

雨水的 k 大約是 4/3，我們之後就會假設 k = 4/3 ，然後我們可以把 beta 寫為 alpha 的函數

我們提一下偏差角是太陽光的向量轉到觀察者向量的角度，然後我們可以立刻得出偏差角（angle of deviation） D(alpha)

我們假設光是均勻的射入角 alpha 平均 0~90 度，下面給出 python 程式

import math
import numpy as np
from typing import List
import matplotlib.pyplot as plt


def observe_angle_deviation(a: float, k: float = 4/3) -> float:
    """
    Calculate the observation angle based on the incident angle and refraction index.
    
    Parameters:
    a (float): Incident angle in degrees.
    k (float): Refraction index. Default is 4/3.
    
    Returns:
    float: Observation angle of deviation in degrees.
    """
    # Convert angle to radians
    a_rad = math.radians(a)
    
    # Calculate refraction angle
    refraction_angle = lambda a, k : math.asin(math.sin(a) / k)
    b_rad = refraction_angle(a_rad, k)
    
    # Calculate observation angle deviation
    d_a_rad = math.pi + 2 * a_rad - 4 * b_rad 
    return d_a_rad / math.pi * 180

# Generate incident angles uniformly distributed between 0 and 90 degrees
incident_angles: np.ndarray = np.linspace(0, 90, 10**5)
observed_angles_deviation: List[float] = [observe_angle_deviation(a) for a in incident_angles]

# Create a figure with two subplots side by side
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 6))

# Plot the relationship between incident angles and observation angles
ax1.plot(incident_angles, observed_angles_deviation)
ax1.set_xlabel('Incident Angle (degrees)')
ax1.set_ylabel('Viewing Angle Deviation (degrees)')
ax1.set_title('Relation between Incident and Viewing Angles Deviation')
ax1.grid(True)

# Create a histogram of the observation angles
ax2.hist(observed_angles_deviation, bins=30, edgecolor='black')
ax2.set_xlabel('Viewing Angle Deviation(degrees)')
ax2.set_ylabel('Frequency')
ax2.set_title('Distribution of Viewing Angles Deviation')
ax2.grid(True)

# Show the plots
plt.tight_layout()
plt.show()

可以得出兩張圖，第一張圖表示入射角與偏差角的關係可以發現 D(alpha) 的極小值大概是 130多度靠近140度，右邊那張圖可以發現發生在極小值的時候會是反射光最集中的地方，有這兩個觀察我們用微分精確的去估出更精確的角度。

import math
import numpy as np
from typing import List
import matplotlib.pyplot as plt


def observe_angle_deviation(a: float, k: float = 4/3) -> float:
    """
    Calculate the observation angle based on the incident angle and refraction index.
    
    Parameters:
    a (float): Incident angle in degrees.
    k (float): Refraction index. Default is 4/3.
    
    Returns:
    float: Observation angle of deviation in degrees.
    """
    # Convert angle to radians
    a_rad = math.radians(a)
    
    # Calculate refraction angle
    refraction_angle = lambda a, k : math.asin(math.sin(a) / k)
    b_rad = refraction_angle(a_rad, k)
    
    # Calculate observation angle deviation
    d_a_rad = 2 * a_rad - 6 * b_rad + 2 * math.pi
    return d_a_rad / math.pi * 180

observed_angles_deviation: List[float] = [observe_angle_deviation(a) for a in incident_angles]

# Create a figure with two subplots side by side
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 6))

# Plot the relationship between incident angles and observation angles

ax1.set_xlabel('Incident Angle (degrees)')
ax1.set_ylabel('Viewing Angle Deviation (degrees)')
ax1.set_title('Relation between Incident and Viewing Angles Deviation')
ax1.grid(True)

import math
k = 4/3

# 這個是極值讓 D(a) 最小 (71.83197124269303)
a_rad = math.acos(math.pow((math.pow(k, 2) - 1) / 8, 0.5))
print(a_rad * 180 / math.pi)
refraction_angle = lambda a, k : math.asin(math.sin(a) / k)
b_rad = refraction_angle(a_rad, k)
d_a_rad = 2 * a_rad - 6 * b_rad + 2 * math.pi 
# 逆時鐘偏差角 D(a) (230.97809352567566)
print('逆時鐘偏差角：', d_a_rad * 180 / math.pi)
# 順時鐘偏差角 360 - D(a) (129.02190647432434)
print('順時鐘偏差角：', 360-(d_a_rad * 180 / math.pi))
# 觀查角度 (50.97809352567566)
print('觀查角度：', 180-(360-(d_a_rad * 180 / math.pi)))

但是如果我們要計算光轉到觀察者的角度用順時鐘轉就要用順時鐘轉，之前的偏差角我們是用逆時鐘，一個逆時鐘一個順時鐘角度就差了360，360 - 230.97809352567566 ~ 129 度。

好了算到現在我們可以得出結論了，觀察者看到霓大概的要抬頭 51 度（= 180 - 129），我們一樣能看紅光跟紫光的觀察者角度，大概是紅光 50.58 跟 紫光 53.6度，所以霓對觀察者來說紅光比紫光低因為 50.58 < 53.6。
